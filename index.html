The web is made up of various pages of htlm. HTML, which stands for HyperText Markup Language is what "glues" the web together and makes up the main type of document on the web.

The internet communication begins with a browser, which communicates to the web using hyperlinks. Those hyperlinks via HTTP, communicate to web servers that provide the information back to the browser and ultimately, to the user. HTTP is the protocol for how the internet communicates with the servers.

As previously mentioned, HTML documents are the heart of the web. It is made up of text content (what you see), markup (what it looks like), references to other documents (images and video), and links to other pages.

Markup are facets of HTML which are made up of tags (tag name) and tag contents. This is called an element. Another way to think of this is Opening Tag (tag) + Content + Closing Tag (/tag) = Element.

Line Breaks

In order to break text into multiple lines, you must use one of two tags, the BR tag or the P tag. The BR tag is an inline tag which breaks text into two lines. It is also a void tag, meaning you only require one tag. The BR tag would look like this: (br).
The P tag, which stands for paragraph is a block element and requires a closing tag. The P tag looks like this (p) Text (/p)

A p tag is a block text because an invisible square surrounds the content within that element. The square has height and width, unlike the inline, br tag which just wraps text to a new line.

Container elements

Container elements consist of SPAN and DIV. SPAN is an inline element while DIV is a block element. They contain their content and can allow us to manipulate how certain content looks. These elements will allow us to format the look of the text.

HTML Document Structure

An HTML Document is structured to include several tags and elements. The structure starts with a DOCTYPE, which is the html string. There are also opening (html) and closing (/html) tags that encapsule the document. The following tag is a head tag (head) (/head) which surrounds the metadata and other information. In between the head tags we have the title tags. The title tags may also reference any CSS or Java script files for the html document. The body tags encapsule the detail of the document. Through the IPND, the focus will be on the body of the html document.
